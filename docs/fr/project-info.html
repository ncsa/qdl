Information générale du projet

Ce document fournit une vue d'ensemble des divers documents et liens qui font partie des informations générales du projet. Tous ces contenus sont générés automatiquement par Maven pour le compte du projet.

Vue d'ensemble

Document Description
Informations de dépendance Ce document présente la manière d'ajouter ce projet en tant que dépendance avec différents outils de gestion des dépendances.
Gestion de la distribution Ce document fournit des informations sur la gestion de la distribution du projet.
Licence du projet C'est un lien à la licence du projet.
Gestion des plugins Ce document liste les plugins définis par pluginManagement.
Plugins du projet Ce document liste les plugins de type build et rapport utilisés par ce projet.
Dépôt de sources C'est un lien au dépôt de sources en ligne qui peut être regardé par l'intermédiaire d'un navigateur Web.
Résumé du projet Ce document liste les autres informations reliées à ce projet
Membres de ce projet Ce document fournit des informations sur les membres de ce projet. Ce sont les individus qui ont contribué au projet d'une façon ou d'une autre.
